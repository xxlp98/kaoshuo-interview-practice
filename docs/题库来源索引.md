# 题库来源索引

本项目采用“源 markdown -> 自动构建 JSON”的维护方式。

## 目录

题库源目录：`考研复试题库及答案/`

当前主题文件：

- 本科学习与经历.md
- 研究生与导师.md
- 家乡与家庭.md
- 团队合作经历.md
- 压力、失败与挑战.md
- 兴趣爱好与生活.md
- 个性与自我认知.md
- 观点与其他.md
- 未来规划与职业.md

## 构建命令

在项目根目录运行：

```bash
node scripts/build_exam_bank.js
```

构建后会更新：

- question-bank.json
- answer-bank.json

## 维护建议

- 修改题目后先执行构建，再本地预览。
- 如有离线音频需求，再执行音频生成脚本。
- 避免手工直接编辑 question-bank.json / answer-bank.json，以免后续被脚本覆盖。
